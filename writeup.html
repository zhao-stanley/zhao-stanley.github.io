Chat App Writeup

Four actions: create, join, send, star.

What / Where / Who

Create Chat

  • What: new chat with a random channel.
  • Where: the chat's own channel (not class-wide, so chats aren't publicly browseable).
  • Who: public within the channel.
{
  activity: "Create",
  type: "Chat",
  title,
  channel: crypto.randomUUID(),
  published: Date.now()
}

Join Chat

  • What: record that I joined a chat. Triggered on create, or on pasting an invite ID.
  • Where: my actor ID (for my sidebar) + the chat channel (so others see me).
  • Who: public. Leave = delete the Join.
{
  activity: "Join",
  target: chatChannel,
  title,
  published: Date.now()
}

Send Message

  • What: a message.
  • Where: the chat channel.
  • Who: public in the channel. Unsend = delete the message.
{
  content,
  published: Date.now()
}

Star Message

  • What: a star pointing at a message.
  • Where: the chat channel.
  • Who: public in the channel. Unstar = delete the Star.
{
  activity: "Star",
  object: messageUrl,
  published: Date.now()
}

Notes

  • You join a chat either by creating one or by pasting an invite ID. The Invite button in the chat header copies the channel ID to share.
  • Star is its own object (not a field on the message) so it's toggleable after the fact and anyone in the chat can flag something important.
  • Recap is a filter over messages that have a Star.
  • Yellow means "important," not "mine." Mine is blue, theirs is gray.
  • All discovers use autopoll: true so new stuff streams in without refreshing.
  • User-facing labels: "Chats," "Invite," "Important," "Recap."
